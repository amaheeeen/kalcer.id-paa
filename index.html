ALGORITMA 01

Merge Sort

Digunakan untuk fitur "Top Viral". Kita membutuhkan Stable Sort agar jika skor viralitas sama, urutan jarak terdekat pengguna tetap terjaga.

RECURRENCE RELATION \[ T(n) = 2T\left(\frac{n}{2}\right) + n \]

Membagi masalah menjadi 2 bagian (logaritmik) dan menggabungkannya secara linear.


def merge_sort(arr):
    if len(arr) <= 1:
        return arr
        
    mid = len(arr) // 2
    left = merge_sort(arr[:mid])
    right = merge_sort(arr[mid:])
    
    return merge(left, right)
# Kompleksitas selalu O(n log n)
# Unlike QuickSort worst case O(n^2)

0/1 Knapsack Optimization

Mengubah fitur "Filter Harga" menjadi sistem rekomendasi cerdas.

PERSAMAAN BELLMAN (RECURSIVE STEP)
\[ K[i][w] = \max \left( v_i + K[i-1][w-w_i], \quad K[i-1][w] \right) \]
$v_i$: Skor Viralitas tempat ke-$i$
$w_i$: Biaya rata-rata tempat ke-$i$
$K$: Tabel solusi optimal
$w$: Sisa budget saat ini
