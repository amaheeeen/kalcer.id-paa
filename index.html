Core Engine

Merge Sort

Mengurutkan ribuan data tempat viral berdasarkan skor tren. Menggunakan pendekatan Divide and Conquer dengan kompleksitas O(n log n) agar sistem tetap responsif di bawah 3 detik.

Dijkstra

Menghitung rute tercepat dari posisi Anda ke Kurasu atau M Bloc. Menghindari "jebakan" macet Senopati dengan kalkulasi bobot graf real-time.

Dynamic Programming

Algoritma cerdas untuk fitur "Filter Budget". Menyelesaikan Knapsack Problem untuk memberikan kombinasi tempat hangout maksimal tanpa bikin dompet jebol.
